## Project Overview

Sillah (صلة) is a preventive health management system designed to help families track hereditary cardiac risks.  
The system allows users to:

- Record family medical history  
- Detect hereditary cardiac patterns  
- Receive automated alerts  
- Book clinic appointments  
- Access educational awareness content  
- View preventive health materials  

This project follows the complete SE201 software engineering lifecycle:
Project Proposal → Requirements → Design → Prototype → Testing → Deployment

------------------------------------------------------------

## Key System Features

### Java Prototype (Console Version)
- Menu-based command-line interface  
- Add and manage family members  
- Automated hereditary risk detection  
- Appointment booking simulation  
- Awareness Hub content  
- Alerts and reminders  

### Web Application (Live Version)
- Responsive interface  
- User registration and login  
- Family member management  
- Appointment scheduling  
- Awareness Hub  
- Alerts and notifications  
- Deployed on Vercel

------------------------------------------------------------

## System Architecture Summary

The system uses a three-layer architecture:

1. Presentation Layer  
   Command-line interface (Java) and dynamic web UI (Vercel)

2. Business Logic Layer  
   Risk detection, booking manager, alerting, medical event processing

3. Data Layer  
   Java in-memory structures (prototype) and structured web-state data (web version)
